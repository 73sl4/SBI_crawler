# SBI_crawler
SBI Crawler using phantomjs

Instructions:

1. Install phantomjs
2. Clone repo
3. Install moment ```npm install moment```
4. Usage
```
phantomjs sbi.js <username> <password> <number_of_months_in_multiple_of_6> 
Options: -v : debug
```
